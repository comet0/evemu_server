Incoming packets

Incoming packets are a PyObject with a PyTuple containing 9 elements.
The first element is an int containing the packet type defined in the MACHONETMSG_TYPE enum.
The second element is an object containing the source address (PyAddress).
The third element is the destination address (PyAddress).
The fourth element is the userID.
The fifth element is the payload as a PyTuple.
The sixth element is a dictionary of named elements (PyDict).
The last three elements are PyNone.

Call Requests

The payload is a single element PyTuple containing a PyTuple that contains 2 elements.
The first value is a PyInt that is currently an unknown and unused value.
The second value is a PyTuple that contains 4 elements decoded as a PyCallStream.
The first element is eight an int of string, and is decoded into either remoteObject or remoteObjectStr values.
The second element is the method name.
The third element is the argument list decoded into arg_tuple.
The fourth element is PyDict containing key value pairs decoded into arg_dict.

Calling a service

When a PyService is called the PyService->Call function is passed the method name, arg_tuple and arg_dict values from the PyCallStream.
A method MachoResolveObject return the nodeID (PyInt).
A method MachoBindObject the arg_tuple is a 2 element tuple decoded into CallMachoBindObject. The first tuple is the bindParams the second is an optional call parameters as a callName, arguments and argument dictionary.
