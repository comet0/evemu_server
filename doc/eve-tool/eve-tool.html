Usage of eve-tool

eve-tool executes commands from a set of command files. Then waits for a file to be specified from 'stdin'. The available commands are.
command description
destiny Converts given string to binary and dumps it as destiny binary.
crc32 Computes CRC-32 checksum of given arguments.
exit Quits current session.
help Lists available commands or prints help about specified one.
now Prints current time in Win32 time format.
obj2sql Converts specified cache object into an SQL update.
script Loads input from specified file(s).
time Interprets given integer as Win32 time.
tri2obj Dumps specified TRI file.
unmarshal Converts given string to binary and unmarshals it.
xstuff Dumps specified STUFF file.

command: xstuff

Usage: [-O <outputPath>] [.stuff file] Extract files from the cpp stuff files. Example: xstuff CCP/EVE/resDX9MinmatarShips.stuff Would extract all the DX9 files from the Minmatar ships file.
